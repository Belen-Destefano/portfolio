Portfolio

React JS

Tecno Stock

App e-commerce de tienda de productos de electrónica. Desarrollado con ReactJs. Funcionalidades: selección de productos según sus categorias. Una vez realizada la compra se modifica el stock de la base de datos de firebase y se obtiene un numero de orden que permite la búsqueda posterior de compras realizadas mediante un input en el carrito. Contiene: react router dom, context, conditional rendering, promesas; Firestore database de Firebase utilizado como base de datos vía Api y para la realizacion de categorias dinamicas de los productos; materializecss, react-toastify.

Stone & Wood

Sitio web creado como proyecto final para Javascript. Fue publicado en el sitio web de CoderHouse, donde realice los estudios, por quedar dentro del 10% de los estudiantes más destacados de la cursada. Desarrollado con Javascript, HTML y CSS.
